# MSME-Loan-App-Docs

## Overview

This repository presents the complete functional and workflow documentation for the **MSME Loan Pre-Qualification Mobile Application**, designed and documented by **Geetanjali Dhoke**.

The documents demonstrate a full end-to-end business analysis approach, covering requirement definition, user journeys, system workflows, and wireframes for a fintech lending solution. The focus of this project is to enable small business owners to check loan eligibility, upload documents digitally, and track application status in real time.

The repository highlights my ability to translate business needs into structured functional documentation, design clear workflows, and align business, technology, and compliance requirements in an Agile delivery environment.

## Structure of the Documentation

Each document in this repository generally includes:

* **Project Overview** – Business context, domain, and application goals
* **Approach Note** – Delivery methodology, security considerations, and assumptions
* **Information Architecture** – Core modules and system components
* **Business & Functional Requirements** – BRDs, FRDs, and non-functional requirements
* **Workflows** – User registration, eligibility check, document upload, and tracking flows
* **Wireframes** – Mobile app screens and admin dashboard views
* **Risk & Constraints** – RAID matrix and mitigation strategies

## Key Workflows Included

The documentation covers the following major workflows:

* **User Registration & OTP Verification Workflow**
  OTP-based onboarding with validation, error handling, and retry logic

* **Loan Eligibility Check Workflow**
  PAN, GSTIN, financial data capture, credit score integration, and eligibility calculation

* **Document Upload & Verification Workflow**
  Secure multi-document upload and verification initiation

* **Application Tracking Workflow**
  Real-time status updates, dashboard visibility, and user notifications

## Wireframes Included

The wireframes illustrate:

* User registration and OTP verification screens
* Financial eligibility input screens
* Document upload interface
* Results display for loan eligibility
* Admin dashboard for monitoring applications and analytics

These wireframes help visualize the end-user journey and system interactions clearly for design and development teams.
